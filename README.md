# 项目更新说明 (Update Log)

## 🚀 近期更新 (Recent Changes)

* **✨ 新增：电脑性能监控功能**
  - 实现了对电脑运行状态的实时监控，能够在屏幕上直观地展示核心硬件指标（如 CPU 使用率、内存占用、温度等）。
* **🛠️ 优化：按键交互逻辑**
  - 重构了底层的按键响应逻辑，改善了防抖处理，提升了按键操作的灵敏度、准确性与整体手感。
